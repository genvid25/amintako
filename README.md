# Генерация кадров мультфильма — пилот

## Что это

Система, которая по вашей раскадровке генерирует картинку для каждого кадра
через нейросеть Google (Gemini, «nano banana»), держит персонажей и локации
одинаковыми по референсам и собирает готовые кадры в удобный дашборд, где
их можно одобрять или отправлять на переделку одной кнопкой.

Сейчас это **локальный пилот**: всё работает на вашем Mac, кадры генерируются,
когда вы запускаете короткую команду. Позже те же самые файлы переедут в облако,
и генерация пойдёт по расписанию сама, а дашборд станет доступен команде по ссылке.

```
ПИЛОТ (ваш Mac, вручную команда)   →   ОБЛАКО (по расписанию, ссылка для команды)
   те же файлы, тот же дашборд, та же логика — просто переезжает наверх
```

## Что нужно сделать один раз

**1. Ключ доступа к нейросети — для пилота уже сохранён.**
   - Ключ уже лежит в файле `.env` в папке проекта. Для теста ничего получать
     и вводить не нужно — скрипт берёт ключ оттуда сам.
   - Если понадобится свой ключ или замена (ротация): откройте **aistudio.google.com**,
     нажмите **Get API key** (Создать ключ), скопируйте строку и замените ею значение
     в файле `.env` (строка `GEMINI_API_KEY=...`). Никому ключ не показывайте.
   - Оплата: генерация платная (см. «Сколько стоит»). В подписке Google **Ultra**
     каждый месяц даётся $100 кредитов — их нужно один раз активировать в разделе
     Billing/Plan для вашего проекта.

**2. Положить референсы персонажей и локаций.**
   - В папке **Референсы** лежит файл `ПРОЧТИ.txt` — там сказано, что положить.
   - Для пилота (сцена 7) нужны 3 файла: `Тако.png`, `Рахим.png`, `лодка_море.png`.
   - Имена файлов должны совпадать с тем, что указано в кадрах (папку менять не нужно).

**3. Проверить, что есть Python.**
   - Он уже встроен в macOS. Откройте приложение **Терминал** и введите:
     ```
     python3 --version
     ```
   - Если показалась версия (например `Python 3.9.6`) — всё готово. Ставить ничего не надо.

> Как открыть Терминал: Cmd+Пробел → напишите «Терминал» → Enter.
> Все команды ниже выполняются в папке проекта. Чтобы перейти в неё, введите:
> ```
> cd ~/Projects/автоАминТако
> ```

## Тест за 3 шага

Выполните в Терминале по очереди (папка проекта, референсы на месте):

**Шаг 1. Ключ.** Уже сделано: ключ сохранён в файле `.env` в корне проекта,
скрипт подхватывает его сам — задавать ничего не нужно.
(Если когда-нибудь смените ключ — просто откройте `.env` и замените строку
`GEMINI_API_KEY=...` на новую.)

**Шаг 2. Сгенерировать первые 3 кадра:**
```
python3 scripts/generate.py --limit 3
```
Скрипт по-русски покажет, что делает, и сохранит картинки в папку `images`.
Хотите сначала посмотреть план без генерации и без ключа — добавьте `--dry-run`:
```
python3 scripts/generate.py --limit 3 --dry-run
```

**Шаг 3. Открыть дашборд:**
```
python3 scripts/serve.py
```
Затем откройте в браузере **http://localhost:8787**.
Готовые кадры появятся карточками. Кнопки на каждой карточке:
- **Одобрить** — кадр принят.
- **Переделать** — впишите, что поправить, и нажмите «Отправить агенту».
  Комментарий сохранится, и на следующем запуске Claude перепишет промт и перегенерирует кадр.
- **Скачать PNG** — сохранить картинку себе.

Дашборд обновляется сам каждые несколько секунд — можно держать его открытым,
пока идёт генерация. Тему (тёмная/светлая) переключает кнопка справа вверху.
Чтобы остановить сервер — вернитесь в Терминал и нажмите **Ctrl+C**.

## Сколько стоит

| Что | Цена |
|-----|------|
| Один кадр (режим **pro**, лучшее качество) | ~ **$0.134** |
| Один кадр (режим **flash**, дешевле) | ~ $0.067 |
| Вся серия целиком (≈137–178 кадров + переделки) | ≈ **$25–31** |

В подписке **Ultra** даётся **$100 кредитов в месяц** — одной серии хватает
с запасом. Дешёвый режим включается флагом `--model flash`, если нужно сэкономить
на черновых прогонах.

## Что дальше

Это пилот, чтобы вы увидели весь цикл вживую на одной сцене. Дальше подключается Claude:

- **Облачная routine** — та же генерация переедет в облако и будет идти по расписанию
  сама, без ручных команд. Правила для агента уже описаны в файле `АГЕНТ.md`.
- **Публичный дашборд** — та же страница станет доступна команде по ссылке,
  с разделами по сотрудникам и сериям.
- **Автоматическая очередь из раскадровки** — новые кадры будут добавляться из XLS-файла
  раскадровки автоматически.

Всё это — задача Claude; трогать код руками вам не придётся.
